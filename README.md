# alx-system_engineering-devops

**General:** Understand RTFM, Shebang, Shell, Terminal vs. Shell, Shell Prompt, and basic History usage.

**Navigation:** Learn how to use cd for directory navigation, pwd for the current directory, and ls for listing files. Discover the . and .. shortcuts, working directories, the root directory, and user home directories.

**Looking Around:** Explore ls, less, and file commands, plus how to use options and arguments.

**A Guided Tour:** Dive into the ln command, common system directories, symbolic and hard links, and the differences between them.

**Manipulating Files:** Master cp, mv, rm, and mkdir commands, and understand wildcard usage.

**Working with Commands:** Learn about type, which, help, and man commands, aliases, and when to use help versus man.

**Reading Man Pages:** Get the most out of man pages, understand section numbers, and navigate through them.

**Keyboard Shortcuts for Bash:** Discover common Bash keyboard shortcuts for efficiency.
